## What is BizCashFlow?

A web-based analytics platform that helps Ghanaian SMEs understand their finances across all payment channels — MTN MoMo, Cash, Vodafone Cash, Bank Transfer, POS, and Credit Sales.

## Features

- **Account Creation & Business Profile** — Industry, location, payment channels, goals
- **Multi-Source Connectors** — Google Sheets, Excel, SQL Database, Payment APIs
- **Smart Schema Mapping** — Auto-detects columns even in Twi
- **Data Cleaning** — Fixes inconsistencies, duplicates, missing values
- **Analytics Dashboard** — Revenue, expenses, channels, peak hours heatmap
- **Cash Flow Forecasting** — 14-day projections with alerts
- **Period Comparison** — This month vs last month
- **Debt/Credit Tracker** — Track who owes you, send WhatsApp reminders
- **Revenue Goals** — Set targets and track progress
- **Auto-Discovered Insights** — AI finds patterns in your custom data fields
- **WhatsApp Summaries & PDF Reports**
- **AI Business Chat** — Ask questions about your data

## Status

This is a **frontend prototype** with synthetic data. Backend development is next.
